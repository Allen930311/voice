### 🚨 問題與解法 (Troubleshooting)
> 🐛 **遇到困難**：Claude Code `settings.json` 不支援 `mcpServers` 欄位（schema validation 失敗）。
> 💡 **解決方案**：MCP server 需註冊在 `~/.claude.json` 而非 `settings.json`，使用 `claude mcp add --scope user` CLI 命令自動寫入正確位置。
